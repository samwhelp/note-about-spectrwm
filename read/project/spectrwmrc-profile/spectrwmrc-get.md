### fzf

執行下面指令，提供fzf列表選項安裝

``` sh
spectrwmrc-theme-get fzf
```

### all

執行下面指令，安裝以下所有的「theme」

``` sh
spectrwmrc-theme-get all
```

### dracula_rounded

執行下面指令，單一安裝「theme / [dracula_rounded](https://github.com/AethanFoot/spectrwm-theme-dracula-rounded/)」

``` sh
spectrwmrc-theme-get dracula_rounded
```
