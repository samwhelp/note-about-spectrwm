# 其他視窗操作


## 關閉視窗

| 按鍵組合 | 功能 |
| --- | --- |
| `Win + q` | 將目前聚焦的視窗關閉 |


## 全螢幕

| 按鍵組合 | 功能 |
| --- | --- |
| `Win + f` | 將目前聚焦的視窗，切換成全螢幕 |


## 交換到主要視窗位置

| 按鍵組合 | 功能 |
| --- | --- |
| `Win + m` | 將目前聚焦的視窗，交換到主要視窗位置 |


## 在浮動模式和平鋪模式之間切換

| 按鍵組合 | 功能 |
| --- | --- |
| `Win + n` | 將目前聚焦的視窗，在浮動模式和平鋪模式之間切換 |


### 最大化

| Key | Description |
| --- | --- |
| `Win + w` | 將目前聚焦的視窗最大化 |


### 最小化

| Key | Description |
| --- | --- |
| `Win + x` | 將目前聚焦的視窗最小化 |
| `Win + Shift + x` | 將目前聚焦的視窗從最小化恢復顯示 |
