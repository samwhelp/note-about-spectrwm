# 其他視窗操作


## 關閉視窗

| 按鍵組合 | 功能 |
| --- | --- |
| `Win + q` | 將目前聚焦的視窗關閉 |


## 全螢幕

| 按鍵組合 | 功能 |
| --- | --- |
| `Win + f` | 將目前聚焦的視窗，切換成全螢幕 |


## 放到主要位置

| 按鍵組合 | 功能 |
| --- | --- |
| `Win + b` | 將目前聚焦的視窗，放到主要Layout位置 |


## 從Floating切換到Tile

| 按鍵組合 | 功能 |
| --- | --- |
| `Win + v` | 將目前聚焦的視窗，從Floating切換到Tile |


### 最大化

| Key | Description |
| --- | --- |
| `Win + w` | Window Maximize |

### 最小化

| Key | Description |
| --- | --- |
| `Win + x` | Window Iconify |
| `Win + Shift + x` | Window Uniconify |
